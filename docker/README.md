# local-dev
inspired of [laradock](https://github.com/laradock/laradock) for simple local development.

## Stater
- create .env file      
just copy from `.env.example` and modify it.    
```shell
cp .env.example .env
```

- run command        
```shell
# mongo mysql redis nginx
docker-compose up -d <which_container_name>
```
